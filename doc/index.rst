nna_methods User's Guide
===================

`nna_methods` is a python tool for neighbor averaging techniques.

The key value of `nna_methods` is to provide scikit-learn style interfaces to
nearest neighbor, voronoi neighbor, and gridded model output statistic (GMOS)
algorithsm. The techniques are used in a variety of scientific fields.


Getting Started
---------------

The best way to get started is to install (see below) and then explore the
examples.


Installation
------------

Right now, you can get the latest version of `nna_methods` with this command.

.. code-block::

    pip install git+https://github.com/barronh/nna_methods.git
